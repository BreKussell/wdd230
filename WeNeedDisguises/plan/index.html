Typography

The website will be using the font "Lucida Handwriting" for any heading on the page. This is chosen to match the font type on the logo. This will keep a consistent look throughout the entire website.

Large heading, for things such as page titles.

Medium heading, for things such as section titles

Small heading, for words that might need emphisiss

The website will be using the font "Arial" for any paragraph font. This was chosen because it is's simple and professional look to it. This was also chosen to match the 'Cosplay' font on the logo to keep a consistent look throughout the website.
